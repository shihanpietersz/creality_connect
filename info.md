# Creality Connect

Home Assistant integration for Creality 3D printers (K1, K1 Max, K1C).

## Features

- **20+ Sensors**: Real-time monitoring of temperatures, progress, position, fans, and layers
- **Full Control**: LED lights, pause/resume, fan speeds, temperature settings
- **Live Media**: Webcam stream and print preview thumbnails
- **Real-time Updates**: Instant sensor updates via native Creality WebSocket protocol

## Requirements

- Home Assistant 2023.1 or later
- Creality printer with Moonraker/Klipper (K1, K1 Max, K1C)
- Printer connected to your local network

## Installation

1. Install via HACS (Home Assistant Community Store)
2. Restart Home Assistant
3. Go to Settings → Devices & Services → Add Integration
4. Search for "Creality Connect"
5. Enter your printer's IP address
6. Configure entities as needed
